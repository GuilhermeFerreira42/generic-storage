## ⚠️ O que precisa de ajuste antes de rodar

### 1. Aliases de path (`@/`)

Os testes frontend usam `@/lib/store`, `@/components/ide/...` etc. Você precisa confirmar que o `vitest.config.ts` tem o alias correto apontando para a raiz do seu projeto:

```typescript
// vitest.config.ts
resolve: {
  alias: {
    '@': path.resolve(__dirname, '.'), // ajuste se necessário
  },
},
```
