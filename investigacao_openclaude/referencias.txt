OBSERVAÇÕES ADICIONAIS:
-----------------------
- O repositório openclaude-main/ é um fork ativo do claude-code upstream
- A estrutura de plugins segue o padrão canônico do Claude Code v1.0+
- Hooks usam comunicação via stdin/stdout com processos shell filhos
- Skills podem ser invocadas manualmente ou auto-trigger via descrição
- Agents suportam isolamento total via Git worktrees quando isolation: worktree
- MCP servers são declarativos via .mcp.json, sem necessidade de código extra no plugin
